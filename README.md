Getting Started:

$ make
$ echo "main() { print("hello, world\n"); }" > hello.lrx
$ ./lorax -c < hello.lrx
$ g++ hello.cpp
$ ./a.out
$ hello, world
$

Running Tests:

$ make
$ ./testall.sh
$
